# Provenance Convention

Every official release of this starter kit should include:

1. A manifest.
2. File hashes.
3. A canonical website URL.
4. A canonical GitHub release.
5. A license.
6. A citation file.
7. Optional online verification.

Normal use must not require network access.

Verification should answer:

1. Is this local package intact?
2. Does it match the bundled manifest?
3. If online, does it match the canonical manifest?
4. If modified, which files changed?
